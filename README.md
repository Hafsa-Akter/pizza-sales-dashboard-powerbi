# 🍕 Pizza Sales Dashboard (Power BI)

## 📊 Overview
This project is an interactive Power BI dashboard analyzing pizza sales performance.

## 🔧 Tools Used
- Power BI
- DAX
- Data Modeling

## 📈 Key Features
- Revenue, Profit, and Profit Margin analysis
- Top 5 pizzas by turnover
- Monthly sales trends
- Peak ordering hours
- Channel performance (Dine-in vs Takeaway)

## 💡 Business Insights
- Identifies high-performing and low-profit products
- Highlights seasonal demand patterns
- Supports operational and marketing decisions
